一.定位: 第一种:绝对定位 position:absolute 1.相对于已经定位的父级元素定位 2.如果父级元素没有定位,就以html根文档定位 3.完全脱离文档流,不占位置 4.通过bottom left right top定位 5.行内元素变成行内块元素,支持宽高 第二种:相对定位 position:relative 1.相对于自身默认位置 2.不脱离文档流,占用位置 3.主要作用使给绝对定位当爹使用,当参照物使用 4.不会改变元素类型 第三种: position:fixed 1.参照物是屏幕可视区 2.元素会变成块元素 3.完全脱离文档流 第四种: position:static 默认的状态,设置于不设置一样 总结:绝对定位和相对定位区别 1.参照物不同 2.相对定位不会对布局造成影响 3.主要作用使给绝对定位当爹使用 4.绝对定位改变类型,相对定位不会 二.元素层级问题 z-index:value越大,越往外 普通元素 < 浮动元素 < 定位元素(position) < z-index:1 子级高于父级,后写的高于先写的
